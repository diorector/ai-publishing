================================================================================
AI-Publishing 사용 가이드
================================================================================

📚 주요 기능
  1. PDF 번역 (translate_pdf.py)
  2. 문서 편집 (edit_document.py)

================================================================================
🚀 빠른 시작
================================================================================

1. 설치
   pip install anthropic pdfplumber python-dotenv
   echo "ANTHROPIC_API_KEY=your-key" > .env

2. PDF 번역
   python translate_pdf.py input/book.pdf
   → 결과: output/output_book_translated.md

3. 문서 편집
   python edit_document.py output/output_book_translated.md
   → 결과: output_edited/output_book_translated/output_book_translated_edited.md

================================================================================
📝 번역 (translate_pdf.py)
================================================================================

기본 사용:
  python translate_pdf.py                    # input/laf.pdf 번역
  python translate_pdf.py book.pdf           # input/book.pdf 번역
  python translate_pdf.py /path/to/file.pdf  # 절대 경로

특징:
  ✓ 20년 경력 번역가 페르소나
  ✓ 병렬 처리 (20개 워커)
  ✓ 용어 일관성 보장
  ✓ 실시간 진행률 표시

예상 비용/시간:
  10페이지: ~$0.20 / ~30초
  35페이지: ~$0.70 / ~1분
  100페이지: ~$2.00 / ~3분

자세한 가이드: TRANSLATION_GUIDELINE.md

================================================================================
✏️ 편집 (edit_document.py)
================================================================================

기본 사용:
  python edit_document.py output/output_laf_translated.md

옵션:
  --pass1-only    Pass 1만 실행 (교정만, 윤문 생략)
  --workers 5     병렬 처리 워커 수 (기본: 10)
  --no-diff       비교 리포트 생성 안 함

편집 프로세스:
  Pass 1: 기계적 교정 (맞춤법, 띄어쓰기, 문장부호)
  Pass 2: 창의적 윤문 (문장 구조, 가독성, 리듬감)

예상 비용/시간:
  1,000자: ~$0.05 / ~10초
  10,000자: ~$0.50 / ~1분
  50,000자: ~$2.50 / ~5분

자세한 가이드: EDITING_GUIDE.md

================================================================================
📁 출력 구조
================================================================================

번역 결과:
  output/
  └── output_파일명_translated.md

편집 결과:
  output_edited/
  └── output_파일명_translated/

      └── output_파일명_translated_diff_report.md (변경사항 비교)

================================================================================
🛠️ 문제 해결
================================================================================

API 키 오류:
  echo "ANTHROPIC_API_KEY=sk-ant-..." > .env

패키지 오류:
  pip install --upgrade anthropic pdfplumber python-dotenv

메모리 부족:
  python edit_document.py file.md --workers 3

느린 처리:
  python edit_document.py file.md --workers 15
