# 한국어 글쓰기 우수성 가이드

## 번역체 완전 제거 가이드 - 40가지 패턴

### 1. "~하는 것이다" → "~이다"
**Before**: 이것은 중요한 문제인 것이다.
**After**: 이것은 중요한 문제다.

### 2. "~하는 바이다" → "~이다"
**Before**: 우리의 목표는 성공하는 바이다.
**After**: 우리의 목표는 성공이다.

### 3. "~되어지다" → "~되다"
**Before**: 새로운 시스템이 도입되어졌다.
**After**: 새로운 시스템이 도입되었다.

### 4. "~에 의해" (과도 사용) → 능동형
**Before**: 이 문제는 팀에 의해 해결되었다.
**After**: 팀이 이 문제를 해결했다.

### 5. "~라는 것이 있다" → "~가 있다"
**Before**: 새로운 기회라는 것이 있다.
**After**: 새로운 기회가 있다.

### 6. "~라는 점에서" → "~에서" / "~므로"
**Before**: 빠르다는 점에서 우수하다.
**After**: 빠르므로 우수하다. / 빠른 점이 우수하다.

### 7. "~할 수가 있다" → "~할 수 있다"
**Before**: 성공할 수가 있다.
**After**: 성공할 수 있다.

### 8. "~하게끔" → "~하도록"
**Before**: 성공하게끔 노력했다.
**After**: 성공하도록 노력했다.

### 9. "~에 대해서" → "~에 대해" / "~을"
**Before**: 이 문제에 대해서 논의했다.
**After**: 이 문제를 논의했다.

### 10. "~에 관해서" → "~에 관해" / "~을"
**Before**: 계획에 관해서 설명했다.
**After**: 계획을 설명했다.

### 11. "~와 같은" (과도 사용) → 구체적 표현
**Before**: AI와 같은 기술이 발전했다.
**After**: AI 등 신기술이 발전했다. / AI 기술이 발전했다.

### 12. "~할 것으로 예상된다" → "~할 전망이다" / "~할 것이다"
**Before**: 성장할 것으로 예상된다.
**After**: 성장할 전망이다.

### 13. "~한 것으로 나타났다" → "~한 것으로 드러났다" / "~했다"
**Before**: 효과적인 것으로 나타났다.
**After**: 효과적이었다.

### 14. "~를 가지고 있다" → "~가 있다" / "~를 가졌다"
**Before**: 큰 가능성을 가지고 있다.
**After**: 큰 가능성이 있다.

### 15. "~를 하게 되다" → "~하다"
**Before**: 시작을 하게 되었다.
**After**: 시작했다.

### 16. "~을 통해서" → "~으로"
**Before**: 이 방법을 통해서 해결했다.
**After**: 이 방법으로 해결했다.

### 17. "~을 기반으로 하여" → "~을 바탕으로" / "~기반"
**Before**: 데이터를 기반으로 하여 분석했다.
**After**: 데이터를 바탕으로 분석했다.

### 18. "~에 따르면" (과도 사용) → "~에 의하면"
**Before**: 보고서에 따르면 성공했다.
**After**: 보고서에 의하면 성공했다. (필요시만)

### 19. "~와 관련된" → "~관련" / "~에 관한"
**Before**: 기술과 관련된 문제
**After**: 기술 관련 문제 / 기술 문제

### 20. "~에 있어서" → "~에서"
**Before**: 디자인에 있어서 중요하다.
**After**: 디자인에서 중요하다. / 디자인이 중요하다.

### 21. "~의 경우에는" → "~는"
**Before**: 이 상황의 경우에는 다르다.
**After**: 이 상황은 다르다.

### 22. "~한 측면에서" → "~면에서" / "~로 보면"
**Before**: 효율성 측면에서 우수하다.
**After**: 효율성 면에서 우수하다.

### 23. "~을 중심으로" (과도 사용) → "~를 위주로" / "주로 ~"
**Before**: 기술을 중심으로 발전했다.
**After**: 주로 기술이 발전했다.

### 24. "~의 형태로" → "~형태로" / "~로"
**Before**: 보고서의 형태로 제출했다.
**After**: 보고서로 제출했다.

### 25. "~을 제공하다" → "~를 주다" / 구체적 동사
**Before**: 솔루션을 제공했다.
**After**: 솔루션을 내놓았다.

### 26. "~을 수행하다" → 구체적 동사
**Before**: 분석을 수행했다.
**After**: 분석했다.

### 27. "~을 실시하다" → 구체적 동사
**Before**: 조사를 실시했다.
**After**: 조사했다.

### 28. "~을 진행하다" → 구체적 동사
**Before**: 회의를 진행했다.
**After**: 회의했다. / 회의를 열었다.

### 29. "~을 받다" (과도 사용) → 구체적 동사
**Before**: 승인을 받았다.
**After**: 승인받았다.

### 30. "~의 일부" → "일부" / "~중 일부"
**Before**: 프로젝트의 일부를 완성했다.
**After**: 프로젝트 중 일부를 완성했다.

### 31. "각각의 ~" → "각 ~" / "~ 각각"
**Before**: 각각의 팀이 참여했다.
**After**: 각 팀이 참여했다.

### 32. "다양한 종류의" → "다양한" / "여러"
**Before**: 다양한 종류의 방법을 시도했다.
**After**: 다양한 방법을 시도했다.

### 33. "대부분의 경우" → "대부분" / "보통"
**Before**: 대부분의 경우 성공한다.
**After**: 대부분 성공한다.

### 34. "~의 일환으로" → "~의 일환으로" (필요시만)
**Before**: 개선의 일환으로 도입했다.
**After**: 개선을 위해 도입했다.

### 35. "~의 결과로" → "~한 결과" / "~해서"
**Before**: 노력의 결과로 성공했다.
**After**: 노력한 결과 성공했다.

### 36. "~에 의거하여" → "~에 따라" / "~을 근거로"
**Before**: 규정에 의거하여 처리했다.
**After**: 규정에 따라 처리했다.

### 37. "~에 해당하는" → "~인" / "~에 속하는"
**Before**: 고객에 해당하는 사람들
**After**: 고객인 사람들 / 고객들

### 38. "~으로 인한" → "~로 인한" / "~때문의"
**Before**: 실수로 인한 오류
**After**: 실수로 인한 오류 (OK) / 실수 때문의 오류

### 39. "~을 위한" (과도 사용) → 문맥에 맞게
**Before**: 성공을 위한 노력
**After**: 성공하려는 노력

### 40. "~을 목적으로" → "~하려고" / "~을 위해"
**Before**: 개선을 목적으로 시작했다.
**After**: 개선하려고 시작했다.

## 문장 리듬과 호흡

### 1. 어미 변화로 리듬 만들기

**단조로운 반복**:
```
이것은 중요하다. 
그것도 중요하다. 
저것도 중요하다.
```

**리듬감 있게**:
```
이것은 중요하다.
그것도 마찬가지다.
저것 역시 그렇다.
```

### 2. 문장 길이 조절

**Before** (긴 문장 연속):
```
이 프로젝트는 작년에 시작되었으며 많은 어려움을 겪었지만 
팀원들의 노력으로 성공적으로 완료되었고 좋은 결과를 얻었으며 
다음 프로젝트를 위한 좋은 경험이 되었다.
```

**After** (리듬 있게):
```
이 프로젝트는 작년에 시작되었다. 
많은 어려움이 있었다. 
하지만 팀원들의 노력으로 성공적으로 마쳤다. 
좋은 결과를 얻었고, 다음 프로젝트를 위한 귀한 경험이 되었다.
```

### 3. 강약 조절

**강약 없이**:
```
매우 중요하다. 매우 필요하다. 매우 좋다.
```

**강약 있게**:
```
무엇보다 중요하다. 
꼭 필요하다.
정말 좋다.
```

## 접속사/전환어 활용법

### 1. 순서/열거
- **첫째, 둘째, 셋째**: 항목 나열
- **우선, 다음으로, 마지막으로**: 순서 설명
- **먼저, 그 다음, 끝으로**: 단계 설명

**예시**:
```
우선, 문제를 파악한다.
다음으로, 해결책을 찾는다.
마지막으로, 실행에 옮긴다.
```

### 2. 대조/전환
- **그러나, 하지만**: 반대 의견
- **반면, 한편**: 대비되는 상황
- **그럼에도, 그런데도**: 예상 밖의 결과

**예시**:
```
비용은 높았다. 
하지만 품질은 탁월했다.
```

### 3. 인과관계
- **그래서, 그러므로**: 자연스러운 결과
- **따라서, 그러니**: 논리적 귀결
- **결과적으로, 결국**: 최종 결과

**예시**:
```
시장이 변했다.
그래서 전략도 바꿔야 한다.
```

### 4. 추가/보충
- **또한, 게다가**: 정보 추가
- **더불어, 아울러**: 함께 고려
- **특히, 무엇보다**: 강조

**예시**:
```
성능이 좋다.
게다가 가격도 저렴하다.
```

### 5. 예시/부연
- **예를 들어, 이를테면**: 구체적 예시
- **즉, 다시 말해**: 재진술
- **말하자면, 요컨대**: 요약

**예시**:
```
비용 절감이 중요하다.
예를 들어, 불필요한 지출을 줄이는 것이다.
```

```
자연: 개선점을 반영하다 / 넣었다
```

## 문장 개선 실전 예시

### 예시 1: 장황한 문장

**Before**:
```
이 시스템은 사용자들이 보다 효율적으로 작업을 수행할 수 있도록 
하기 위한 목적으로 개발되어진 것이며, 다양한 기능들을 제공함으로써 
생산성을 향상시키는 데 기여할 수 있을 것으로 기대되고 있다.
```

**After**:
```
이 시스템은 사용자가 효율적으로 일하도록 돕는다.
다양한 기능으로 생산성을 높일 것으로 기대한다.
```

### 예시 2: 번역체 과다

**Before**:
```
결과적으로, 이러한 방법을 통해서 문제를 해결하는 것이 가능한 것으로 
나타났으며, 이는 앞으로의 프로젝트에 있어서 중요한 의미를 가지는 
것이라고 할 수 있을 것이다.
```

**After**:
```
결국, 이 방법으로 문제를 해결할 수 있었다.
이는 앞으로의 프로젝트에 중요한 의미가 있다.
```

### 예시 3: 피동형 과다

**Before**:
```
새로운 전략이 수립되었고, 팀에 의해 실행이 진행되고 있으며, 
긍정적인 결과가 도출되는 것으로 관찰되어지고 있다.
```

**After**:
```
새로운 전략을 세웠고, 팀이 실행하고 있다.
긍정적인 결과가 나오고 있다.
```

### 예시 4: 명확성 부족

**Before**:
```
그것은 그 시점에서 그러한 방식으로 그들에 의해 처리되었던 것이다.
```

**After**:
```
그 문제는 2020년 3월에 프로젝트 팀이 워크샵 방식으로 해결했다.
```

## 체크리스트

### 번역체 제거 체크리스트
- [ ] "~하는 것이다", "~하는 바이다" 제거
- [ ] "~되어지다" → "~되다"
- [ ] "~에 의해" 최소화 (능동형 선호)
- [ ] 불필요한 "~에 대해서", "~에 관해서" 제거
- [ ] "~을 가지고 있다" → "~이 있다"

### 자연스러운 표현 체크리스트
- [ ] 명사+하다 → 고유 동사 활용
- [ ] 피동 → 능동 전환
- [ ] 구체적 동사 사용
- [ ] 한국어다운 접속사 사용
- [ ] 리듬감 있는 문장 배열

### 독자 친화성 체크리스트
- [ ] 문장 길이 적절 (40-60자)
- [ ] 전문 용어 설명
- [ ] 구체적 예시 포함
- [ ] 논리적 흐름
- [ ] 읽기 편한 호흡
