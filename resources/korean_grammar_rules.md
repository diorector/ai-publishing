# 한국어 맞춤법 및 교정 규정

## 국립국어원 한글 맞춤법 핵심 규정

### 제5장 띄어쓰기 (제41항~제50항 요약)

#### 제41항: 조사는 그 앞말에 붙여 쓴다
- 예: 꽃이, 꽃마저, 꽃밖에, 꽃에서부터

#### 제42항: 의존 명사는 띄어 쓴다
- 할 수 있다 (○) / 할수있다 (✗)
- 먹을 만하다 (○) / 먹을만하다 (✗)
- 할 줄 안다 (○) / 할줄안다 (✗)
- 할 뿐이다 (○) / 할뿐이다 (✗)
- 아는 것이 힘이다 (○) / 아는것이 힘이다 (✗)

#### 제43항: 단위를 나타내는 명사는 띄어 쓴다
- 한 개 (○) / 한개 (✗)
- 차 한 대 (○) / 차 한대 (✗)
- 소 한 마리 (○) / 소 한마리 (✗)
- 옷 한 벌 (○) / 옷 한벌 (✗)
- 연필 한 자루 (○) / 연필 한자루 (✗)

**예외**: 순서를 나타내는 경우나 숫자와 어울릴 때는 붙여 쓸 수 있음
- 제1과, 10개, 두시삼십분

#### 제47항: 보조 용언은 띄어 씀을 원칙으로 하되, 경우에 따라 붙여 쓸 수 있다
- 불이 꺼져 간다 (○) / 불이 꺼져간다 (○)
- 그가 올지도 모른다 (○) / 그가 올지도모른다 (○)
- 책을 읽어 보다 (○) / 책을 읽어보다 (○)
- 해 주다, 해 버리다, 해 내다

#### 제48항: 성과 이름, 성과 호 등은 붙여 쓰고, 이에 덧붙는 호칭어는 띄어 쓴다
- 김철수 씨 (○) / 김철수씨 (✗)
- 최영희 박사 (○) / 최영희박사 (✗)

## 자주 틀리는 맞춤법 40선

### 1. 되다/돼다 구분
- **규칙**: "되어"가 줄어든 말이 "돼"
- 되다 (○): 잘 되다, 안 되다
- 돼다 (✗): 잘 돼다 (○), 안 돼 (○)
- **확인법**: "해"로 바꿔보기 (되다→하다, 돼→해)

### 2. 안 되다 / 안되다
- 안 되다 (○): "안"은 부사이므로 띄어쓰기
- 안되다 (✗)

### 3. 할 수 있다 / 할수있다
- 할 수 있다 (○): "수"는 의존명사
- 할수있다 (✗)

### 4. 웬일 / 왠일
- 웬일 (○): "어찌된 일"의 뜻
- 왠일 (✗)
- **참고**: 왠지 (○) - "왜인지"의 줄임말

### 5. 금세 / 금새
- 금세 (○): "금시에"의 줄임말
- 금새 (✗)

### 6. -ㄴ지 / -은지
- **의문**: ~ㄴ지 / ~은지
  - 얼마나 큰지 (○)
  - 언제 오는지 (○)
- **시간**: ~ㄴ 지 / ~은 지 (띄어쓰기)
  - 떠난 지 오래다 (○)
  - 만난 지 5년이다 (○)

### 7. -던지 / -든지
- **-던지**: 과거 회상
  - 먹던지 마시던지 (과거)
- **-든지**: 선택
  - 먹든지 마시든지 (선택)
  - 가든지 말든지 (○)

### 8. 어떻게 / 어떻해
- 어떻게 (○)
- 어떻해 (✗)

### 9. 왠지 / 웬지
- 왠지 (○): "왜인지"의 준말
- 웬지 (✗)

### 10. 며칠 / 몇일
- 며칠 (○)
- 몇일 (✗)

### 11. 얼마 만에 / 얼마만에
- 얼마 만에 (○): "만"은 의존명사
- 얼마만에 (✗)

### 12. 가끔씩 / 가끔식
- 가끔씩 (○): "씩"은 접미사
- 가끔식 (✗)

### 13. 뵙다 / 뵈다
- 뵙다 (○): "보다"의 높임말
- 뵈다 (✗)
- 뵙겠습니다 (○) / 뵈겠습니다 (✗)

### 14. 바라다 / 바래다
- 바라다 (○): 희망하다
- 바래다 (✗)
- **참고**: 바래다 (○) - 색이 바래다 (탈색)

### 15. 다르다 / 틀리다
- 다르다 (○): 같지 않다
- 틀리다 (○): 맞지 않다, 잘못되다
- 너와 나는 다르다 (○)
- 답이 틀리다 (○)

### 16. -랑 / -량
- **-랑**: 조사
  - 너랑 나랑 (○)
- **-량**: 한자어 접미사
  - 분량, 수량 (○)

### 17. 맞히다 / 맞추다
- 맞히다 (○): 정답을 맞히다
- 맞추다 (○): 시간을 맞추다

### 18. 부치다 / 붙이다
- 부치다 (○): 편지를 부치다, 전을 부치다
- 붙이다 (○): 우표를 붙이다

### 19. 가르치다 / 가리키다
- 가르치다 (○): 교육하다
- 가리키다 (○): 손으로 가리키다

### 20. -던 / -든
- **-던**: 과거 회상
  - 살던 집 (○)
- **-든**: 선택, 상관없음
  - 어디든 (○)

### 21. 나름대로 / 나름데로
- 나름대로 (○)
- 나름데로 (✗)

### 22. 로써 / 로서
- **로써**: 수단/도구
  - 칼로써 자르다 (○)
- **로서**: 자격/지위
  - 학생으로서 (○)

### 23. 계시다 / 계신다
- 계시다 (○): "있다"의 높임말
- 계신다 (✗): 잘못된 표현

### 24. 됐다 / 됬다
- 됐다 (○): "되었다"의 준말
- 됬다 (✗)

### 25. 짧다 / 짧따
- 짧다 (○)
- 짧따 (✗)

### 26. 낫다 / 낳다 / 났다
- 낫다 (○): 병이 낫다, 더 낫다
- 낳다 (○): 아이를 낳다
- 났다 (○): 나다의 과거

### 27. 빗다 / 빚다 / 빛다
- 빗다 (○): 머리를 빗다
- 빚다 (○): 떡을 빚다, 빚을 빚다
- 빛다 (○): 빛이 빛다

### 28. 설거지 / 설겆이
- 설거지 (○)
- 설겆이 (✗)

### 29. 왠만하면 / 웬만하면
- 웬만하면 (○)
- 왠만하면 (✗)

### 30. -더라 / -다라
- -더라 (○): 과거 경험 회상
  - 그가 오더라 (○)
- -다라 (✗)

### 31. 반드시 / 반듯이
- 반드시 (○): 꼭, 틀림없이
- 반듯이 (○): 바르고 곧게
  - 반드시 성공하다 (○)
  - 반듯이 누워있다 (○)

### 32. 문안하다 / 무난하다
- 무난하다 (○): 탈이 없다
- 문안하다 (✗)

### 33. 곰곰이 / 곰곰히
- 곰곰이 (○)
- 곰곰히 (✗)

### 34. 일찍이 / 일찌기
- 일찍이 (○)
- 일찌기 (✗)

### 35. 휘날리다 / 휘늘리다
- 휘날리다 (○)
- 휘늘리다 (✗)

### 36. 이따가 / 있다가
- 이따가 (○): 조금 뒤에
- 있다가 (○): 있는 상태에서
  - 이따가 만나자 (○)
  - 집에 있다가 나왔다 (○)

### 37. 던지다 / 던지다
- 던지다 (○): 물건을 던지다
- 던지다 (O): 질문을 던지다

### 38. 예쁘다 / 이쁘다
- 예쁘다 (○): 표준어
- 이쁘다 (✗): 비표준어

### 39. 아무튼 / 아뭏든
- 아무튼 (○)
- 아뭏든 (✗)

### 40. 쌔다 / 세다
- 세다 (○): 힘이 세다
- 쌔다 (✗)

## 외래어 표기법 주요 규칙

### 1. 받침 표기
- 외래어의 끝소리는 'ㄱ, ㄴ, ㄹ, ㅁ, ㅂ, ㅅ, ㅇ'만 사용
- 예: 버스 (○) / 버쓰 (✗)
- 택시 (○) / 택씨 (✗)

### 2. 거센소리 표기
- 파열음 표기에는 거센소리를 쓰지 않음을 원칙으로 함
- 예외: 인명, 지명에서 관용적으로 사용

### 3. 된소리 표기
- 외래어의 1음운은 원칙적으로 1기호로 적음
- 예: 쇼핑 (○) / 쑈핑 (✗)
- 샴푸 (○) / 샴뿌 (✗)

### 4. 영어 인명 표기
- Rand → 랜드 (○) / 랜드 (○)
- Michelle → 미셸 (○)
- Kelly → 켈리 (○)

## 문장부호 사용법

### 1. 마침표(.)
- 서술, 명령, 청유 등을 나타내는 문장 끝에 씀
- 아라비아 숫자만으로 연월일을 표시할 때 씀: 2025.11.18.

### 2. 쉼표(,)
- 같은 자격의 어구를 열거할 때: 빨강, 파랑, 노랑
- 짝을 지어 구별할 때: 원인과 결과, 수요와 공급
- 끼어든 어구 앞뒤에: 철수는, 솔직히 말하면, 거짓말을 했다

### 3. 따옴표(' ', " ")
- 글 가운데서 직접 대화를 표시: "안녕하세요"
- 말이나 글을 직접 인용할 때
- 강조하거나 특별한 뜻을 나타낼 때: 현대의 '정보화 사회'

### 4. 물음표(?)
- 의문문이나 의문을 나타낼 때

### 5. 느낌표(!)
- 감탄문이나 놀람, 느낌 등을 나타낼 때

### 6. 줄표(—)
- 제목 다음에 간단한 부제를 붙일 때
- 앞의 내용을 요약할 때

### 7. 가운뎃점(·)
- 열거할 어구들을 일정한 기준으로 묶어서 보일 때
- 의존 명사 '등'이 쓰일 때: 책·공책 등

## 교정 시 주의사항

### 1. 숫자와 단위
- 한글 단위: 띄어쓰기 - 10 개, 5 명
- 실제로는: 관용적으로 붙여 쓰는 경우가 많음 - 10개, 5명
- **원칙**: 띄어 쓰는 것이 맞으나, 붙여 써도 허용됨

### 2. 한글+영문/숫자
- 반드시 띄어쓰기: 3 개, iPhone 13
- 예외: 고유명사는 원 표기 존중

### 3. 전문 용어
- 해당 분야에서 관용적으로 쓰이는 표기 존중
- 예: 스타트업, 벤처캐피털, CEO

### 4. 번역체 표현 (교정 대상)
- "~되어지다" → "~되다"
- "~에 의해" (과도 사용 시) → 능동형으로 전환
- "~것이다" (불필요한 경우) → 제거
- "~하는 바이다" → "~이다"

### 5. 원문 의미 보존
- 교정은 맞춤법과 띄어쓰기 수정만
- 단어 선택이나 문장 구조는 변경하지 않음
- 저자의 문체와 스타일 존중
