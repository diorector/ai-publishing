## 📋 기본 원칙

### 1. **톤 & 스타일**

**핵심**: 비즈니스 교양서 / 전문적이면서 접근 가능한 / 성인 독자층

```
✅ 적절한 톤:
- 존댓말 통일: "~습니다", "~합니다"
- 문체: 명확하고 직설적이면서도 친근
- 어조: 저자의 개인적 이야기 → 따뜻하고 솔직함
- 리스트/설명: 객관적이고 명확함

❌ 피할 톤:
- 반말 혼재 ("~했다", "~하는데")
- 과도한 경어 (어색함)
- 기술 용어만 사용 (접근성 저하)
- 너무 친근한 문어 체 (비전문적)
```

### 2. **문장 길이 & 구조**

```
권장:
- 평균 문장 길이: 20-30단어
- 각 문단: 3-5문장
- 전환문: 명확한 논리적 연결

피하기:
- 40단어 이상 장문 (가독성 저하)
- 중첩된 문장 (번역 어려움)
- 애매한 대명사 참조
```

## 📝 번역 체크리스트 (각 섹션별)

각 섹션 번역 후 이 체크리스트로 검증:

```
□ 톤: 모든 문장이 존댓말("~습니다")로 통일됨
□ 용어: 위 사전의 용어가 일관성 있게 사용됨
□ 문장: 평균 20-30단어, 읽기 쉬운 길이
□ 논리: 문단 간 전환이 자연스러움
□ 이름: 고유명사(사람/회사명) 유지 또는 일관성 있게 번역
□ 기술 용어: 원문과 일치 (불필요한 번역 피함)
□ 가독성: 어색한 표현 없음
```

## 🎯 부분별 톤 가이드

### A. 개인 일화 섹션 (저자의 개인 경험)

```
톤: 진솔하고 따뜻하면서도 비판적 성찰
예시:
"나는 25세였고, 그의 도착에 완전히 당황했습니다.
하지만 저는 형편없는 거짓말쟁이라서 이렇게 말했습니다:
'죄송하지만 그 사람은 여기 없는 것 같습니다.'"

❌ 피할 것:
"나는 25살이었고, 그 사람이 왔는데 정말 깜짝 놀랐다."
```

### B. 통계/데이터 섹션

```
톤: 객관적이고 명확함, 증거 제시
예시:
"하버드 비즈니스 스쿨의 코호트 분석에 따르면,
초기 단계의 기술 회사 75% 이상이
투자자의 자본을 회수하지 못하고 있습니다."

✅ 명확하고 직설적
```

### C. 조언/교훈 섹션

```
톤: 직설적이면서 실용적, 강한 주장
예시:
"투명성은 처음에는 더 어렵습니다.
하지만 이것은 엄청나게 강력하며,
당신 주변의 모든 사람에게 긍정적인 영향을 미칩니다."

✅ 강하지만 지나치지 않음
```

## 🔄 섹션 간 연결 가이드

### 인접 섹션 연결

**문제**: 현재 각 섹션이 독립적 → 연결 약함

**해결**: 이전 섹션의 주제를 다음 섹션에서 언급

```
예시:
Section 2 끝: "...이 기업가들로부터 배운 것은?"
Section 3 시작: "그들이 내게 가르친 것은 정확히 이것입니다."
→ 자연스러운 흐름

현재 상태:
Section 2 끝: "...스타트업 치트 코드였습니다."
Section 3 시작: "# 한국어 번역" (완전히 끊김)
```

### 3. 문장 길이 & 가독성 (평균 20-30단어)

**가이드라인**: 평균 문장 길이 20-30단어, 각 문단 3-5문장

**구현**:
```python
# translate_full_pdf.py의 translate_with_claude() 함수 내 5가지 번역 철학

【번역 철학 #3: 읽기 쉬운 문장】
- 한 문장에 하나의 핵심 아이디어
- 긴 문장은 2-3개로 분리
- 불필요한 수식어 제거
- 리듬감 있게
```

**실제 예시** (프롬프트에 명시):
```
원문: "That's when I realized we had to pivot."

❌ 나쁜 번역 (번역체, 문장 길이 과다):
"그것은 우리가 피벗을 해야만 했다는 것을 제가 깨달았을 때였습니다."

✅ 좋은 번역 (자연스럽고 짧음):
"그때 깨달았습니다. 방향을 바꿔야 한다는 것을요."
```

**검증 방법**:
- 번역 텍스트에서 각 문장의 단어 수를 세어 평균 계산
- Python 스크립트로 자동 검증:
  ```python
  import re
  sentences = re.split(r'[.!?]+', translated_text)
  word_counts = [len(s.split()) for s in sentences if s.strip()]
  avg_length = sum(word_counts) / len(word_counts)
  print(f"평균 문장 길이: {avg_length:.1f}단어")
  ```
